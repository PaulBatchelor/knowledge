\problem{2.3.34}
%@p 2.3.34
Find the following limit or state that it does not exist:

$$\
\lim_{x \to -5} {\
{x^2 + 7x + 10}
\over
{x + 5}
}
$$


%@s 2.3.34
\solution{2.3.34}
Solution: $-3$.

\problem{2.3.45}
%@p 2.3.45
Find the following limit or state that it does not exist.

$$\
\lim_{x \to 26} {
{x - 26}
\over
{\sqrt{x} - \sqrt{26}}
}
$$

\solution{2.3.45}
%@s 2.3.45
First, simplify the limit to be:

$$\
\lim_{x \to 26} {
\sqrt{x} + \sqrt{26}
}
$$

The limit can be computed to be
$\lim_{x \to 26} {\sqrt{x} + \sqrt{26}} = 2 \sqrt{26}$

\problem{2.3.88}
Suppose:

$$
f(x) = \cases{
-4x + b & if $x \le 2$ \cr
x - 2 & if $x > 2$ \cr
}
$$

Determine a value for the constant $b$ for which
$\lim_{x \to 2} f(x)$ exists and state the value of
the limit, if possible.

\solution{2.3.88}

$b = 8$, $\lim_{x \to 2} f(x) = 0$.
